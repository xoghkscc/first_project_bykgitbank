# POS기 프로젝트
## 0_개요
중 대형 마트에서 사용하고 있는 POS기기를 구현할 목적으로 개발하였다. POS기기를 통해 물품의 계산뿐만이 아니라 재고 관리, 회원 관리, 직원 관리, 배달 관리 등 다양한 역할을 할 수 있도록 개발하였다. 
## 1_개발환경
  * OS : window 10
  * 개발언어 : java
  * DB : oracle
  * Tool : eclipse
  * 외부 라이브러리 : jdbc(ojdbc8.jar), jdatepicker-1.3.2.jar, HikariCP-4.0.3.jar
